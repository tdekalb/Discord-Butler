# Butler Bot

Simple Discord Bot for answering questions made in chat

## Installation

You must have Node v12 installed

In your checkout dir run:

```bash
npm init
npm install discord.js --save
```

## To Start Bot

In your checkout dir run:

```bash
node .
```
